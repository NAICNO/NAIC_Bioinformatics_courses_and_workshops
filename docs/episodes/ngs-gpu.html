Accelerated NGS analysis in AI cloud (GPU cloud)

Objective

Current Next Generation Sequence (NGS) analysis pipelines are run on CPU-platforms and take a long time to complete (e.g., days). In this session, we will discuss how to reduce the processing time of NGS analysis pipelines using GPU-platforms.
