Introduction
============

Next Generation Sequencing
--------------------------

-  DNA Sequencing process

   -  Process of determining the order of nucleotides in a genome
      (genome codes the complete set of instructions for cellular
      functions).

-  Next Generation Sequencing (NGS)

   -  Increasing the throughput by massively parallelising the
      sequencing process
